# 🌱 Smart Irrigation System (Arduino)

Automated plant watering system using Arduino and a soil moisture sensor.

## 📌 Overview

This project is an automatic irrigation system that monitors soil moisture levels and activates watering when the soil is dry. It was developed as a practical project to apply concepts of electronics, programming, and IoT.

---

## ⚙️ How it works

- The soil moisture sensor reads the humidity level
- If the soil is dry, the system activates watering
- LEDs indicate the current state:
  - 🟢 Soil OK
  - 🔴 Soil Dry

---

## 🛠️ Technologies & Components

- Arduino
- Soil moisture sensor
- LEDs
- Relay + Water pump
